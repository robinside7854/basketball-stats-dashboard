# Plan: opponent-defense-analysis
> 수비 코치 관점의 상대팀 공격 심층 분석 대시보드

---

## 1. 현재 상태 진단

### 현재 추적 중인 데이터
| 항목 | 상태 |
|------|------|
| 슈팅 유형별 시도/성공 (3P/미들/레이업/골밑/FT) | ✅ |
| 선수별 PTS / FG% / 3P% / FT% / OR | ✅ |
| 팀 공격 스타일 분포 (라벨 스택 바) | ✅ |
| 쿼터 정보 (events.quarter) | ✅ 기록됨, 미집계 |
| 게임별 드릴다운 | ✅ |
| 영상 타임스탬프 | ✅ |

### 수비 코치 관점의 핵심 공백
| 부재 항목 | 수비 전략 영향 |
|-----------|---------------|
| 쿼터별 공격 집중도 | Q4 마무리 선수, 중요 시간대 매칭 불가 |
| 선수별 팀 내 슈팅 점유율(USG%) | "원맨팀" vs "분산 공격" 판단 불가 |
| 성공률 낮은 유형 자동 하이라이트 | 유도 수비 전략 수립 불가 |
| 게임별 성적 트렌드 | 컨디션 상승/하강 팀 판단 불가 |
| 수비 전략 자동 추천 | 분석을 행동으로 연결하는 가교 없음 |

## 2. 수비 코치가 필요한 분석 5가지

### [분석 1] 공격 집중도 — 도넛 차트
**"이 팀은 원맨 팀인가, 분산 공격인가?"**

팀 전체 슈팅 시도 중 선수별 점유율(USG%)을 도넛 차트로 시각화.
- 상위 1명 점유율 > 40% → **Box-and-1** 전략 권고
- 분산 (각 20%대) → **Zone defense** 고려

```
[도넛 차트]
  #7 민준  34%
  #4 도훈  28%
  #11 재원 22%
  기타     16%
```

**구현**: 기존 데이터로 즉시 계산 가능 (fga 기준 점유율)

---

### [분석 2] 쿼터별 득점 패턴 — 그룹 바 차트
**"이 팀은 어느 쿼터에 폭발하는가?"**

Q1~Q4별 누적 득점 + 슈팅 성공률을 시각화.
이미 events에 quarter 필드가 있으므로 API 집계만 추가하면 됨.

```
Q1  ████████  18점  FG%42%
Q2  ██████    14점  FG%38%
Q3  ██████████22점  FG%51%  ← 주의 쿼터
Q4  ████████  18점  FG%44%
```

- Q3 집중 팀 → 하프타임 후 수비 강도 유지 지시
- Q4 강세 팀 → 클러치 선수 미리 파악, 파울 관리

**구현**: API에 quarter_breakdown 추가 (기존 events 데이터 활용)

---

### [분석 3] 수비 취약 유도 구간 — 자동 하이라이트 카드
**"이 팀을 어디로 몰면 성공률이 떨어지는가?"**

슈팅 유형별 성공률을 분석, 수비 전략 카드 3종 자동 생성.

```
🛡️ 전략 카드 A — 페인트 존 봉쇄
  근거: 레이업+골밑 = 팀 전체 슈팅의 52%
  대응: 빅맨 2명 페인트 존 고정, 드롭 커버리지

🎯 전략 카드 B — 3점 라인 클로즈아웃 불필요
  근거: 3P% 28% (리그 평균 이하)
  대응: 페인트 집중 수비, 3점 라인은 자유 허용

💊 전략 카드 C — Hack-a-Player 대상 없음
  근거: 모든 선수 FT% ≥ 65%
  대응: 의도적 파울 전술 비추천
```

**구현**: 기존 shot_breakdown + fg_pct 데이터로 규칙 기반 자동 생성

---

### [분석 4] 경기별 성적 트렌드 — 라인 차트
**"이 팀이 성장하고 있는가, 약해지고 있는가?"**

경기 날짜순 득점, FG%, 3PA(3점 시도) 추이를 라인 차트로 표시.
- 상승 트렌드 → 다음 경기는 최고 컨디션으로 대비
- 하강 트렌드 → 물리적 압박 수비 통해 혼란 가중

**구현**: 기존 gameDetails 데이터 활용, 프론트에서 라인 차트 렌더링

---

### [분석 5] 레이더 차트 — 팀 공격 성향 프로파일
**"한 눈에 이 팀의 공격 DNA는?"**

5축 레이더: 외곽(3P비율) / 중거리 / 돌파(레이업) / 포스트(골밑) / FT 획득
색 바보다 직관적인 공격 패턴 인지 가능.

```
      외곽(3P)
       ▲
  FT획득 ◄─────► 중거리
       │
돌파 ◄─┤─► 포스트
```

**구현**: SVG 기반 레이더 차트 컴포넌트 (외부 라이브러리 불필요)

## 3. 구현 우선순위

| 우선순위 | 기능 | 난이도 | 신규 DB | 임팩트 |
|---------|------|-------|---------|-------|
| ★★★ | 수비 전략 카드 (분석 3) | 낮음 | 불필요 | 매우 높음 |
| ★★★ | 공격 집중도 도넛 차트 (분석 1) | 낮음 | 불필요 | 높음 |
| ★★☆ | 쿼터별 득점 패턴 (분석 2) | 중간 | 불필요 | 높음 |
| ★★☆ | 경기별 트렌드 라인 (분석 4) | 중간 | 불필요 | 중간 |
| ★☆☆ | 레이더 차트 (분석 5) | 중간 | 불필요 | 중간 |

> **핵심**: 5가지 모두 기존 DB 컬럼 추가 없이 구현 가능

## 5. 기술 스택

- **SVG 레이더 차트**: 외부 라이브러리 없이 직접 구현 (recharts 설치 고려)
- **도넛 차트**: SVG 직접 구현 또는 recharts
- **라인/바 차트**: recharts (이미 Next.js와 호환)
- **전략 카드**: 순수 로직 + JSX (라이브러리 불필요)
- **쿼터 집계**: opponent-stats API에 quarter_breakdown 필드 추가

## 결론

현재 기록 시스템으로 **수비 코치가 실제 사용할 수 있는 인사이트 5가지**를 추가 DB 변경 없이 구현 가능.
가장 임팩트 높은 **수비 전략 카드 + 공격 집중도 도넛 차트**부터 시작 권장.
